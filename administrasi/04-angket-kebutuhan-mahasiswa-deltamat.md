# Angket Kebutuhan Mahasiswa terhadap DELTA-MAT

## Identitas Angket

| Komponen | Keterangan |
| --- | --- |
| Nama angket | Angket Kebutuhan Mahasiswa terhadap DELTA-MAT |
| Fitur | DELTA-MAT |
| Sistem induk | SIMTA - Sistem Informasi Manajemen Tugas Akhir |
| Sasaran responden | Mahasiswa yang sedang mencari, mengajukan, atau menyusun judul tugas akhir |
| Tujuan | Mengidentifikasi kebutuhan mahasiswa terhadap fitur database tugas akhir DELTA-MAT |
| Bentuk | Angket tertutup dengan skala Likert |

## Petunjuk Pengisian

Angket ini digunakan untuk mengetahui kebutuhan mahasiswa terhadap fitur DELTA-MAT. DELTA-MAT adalah fitur untuk menelusuri data judul tugas akhir, melihat indikasi kemiripan judul, membuka detail data, dan mengakses dokumen tugas akhir yang tersedia.

Berikan tanda pada salah satu pilihan skala berikut:

| Skor | Keterangan |
| --- | --- |
| 1 | Sangat Tidak Setuju |
| 2 | Tidak Setuju |
| 3 | Netral |
| 4 | Setuju |
| 5 | Sangat Setuju |

## Interpretasi Hasil

Hasil angket dapat digunakan untuk menentukan prioritas pengembangan DELTA-MAT.

| Rata-rata skor | Interpretasi |
| --- | --- |
| 1.00 - 1.80 | Kebutuhan sangat rendah |
| 1.81 - 2.60 | Kebutuhan rendah |
| 2.61 - 3.40 | Kebutuhan sedang |
| 3.41 - 4.20 | Kebutuhan tinggi |
| 4.21 - 5.00 | Kebutuhan sangat tinggi |

Bagian dengan rata-rata skor tertinggi dapat menjadi prioritas pengembangan DELTA-MAT, terutama pada pencarian judul, analisis kemiripan, halaman detail, dan akses dokumen PDF.
