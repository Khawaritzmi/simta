# Angket Kebutuhan Mahasiswa terhadap SIMTA

## Identitas Angket

| Komponen | Keterangan |
| --- | --- |
| Nama angket | Angket Kebutuhan Mahasiswa terhadap SIMTA |
| Sistem | SIMTA - Sistem Informasi Manajemen Tugas Akhir |
| Sasaran responden | Mahasiswa yang sedang atau akan menempuh proses tugas akhir |
| Tujuan | Mengidentifikasi kebutuhan mahasiswa terhadap aplikasi web SIMTA |
| Bentuk | Angket tertutup dengan skala Likert |

## Petunjuk Pengisian

Angket ini digunakan untuk mengetahui kebutuhan mahasiswa terhadap pengembangan aplikasi web SIMTA. Responden diminta memberikan jawaban sesuai pengalaman, kebutuhan, dan harapan selama menjalani proses tugas akhir.

Berikan tanda pada salah satu pilihan skala berikut:

| Skor | Keterangan |
| --- | --- |
| 1 | Sangat Tidak Setuju |
| 2 | Tidak Setuju |
| 3 | Netral |
| 4 | Setuju |
| 5 | Sangat Setuju |

## Interpretasi Hasil

Hasil angket dapat digunakan untuk menentukan prioritas pengembangan SIMTA.

| Rata-rata skor | Interpretasi |
| --- | --- |
| 1.00 - 1.80 | Kebutuhan sangat rendah |
| 1.81 - 2.60 | Kebutuhan rendah |
| 2.61 - 3.40 | Kebutuhan sedang |
| 3.41 - 4.20 | Kebutuhan tinggi |
| 4.21 - 5.00 | Kebutuhan sangat tinggi |

Bagian dengan rata-rata skor tertinggi dapat dijadikan prioritas utama dalam pengembangan aplikasi web SIMTA.
